# Makefiles

## 0-Makefile
Create the first makefile
Requirements:
  * name of the executable: `school`
  * rules: `all`
    * The `all` rule builds your executable
  * variables: none

## 1-Makefile
Requirements:
  * name of the executable: `school`
  * rules: `all`
    * The `all` rule builds your executable
  * variables: `CC`, `SRC`
    * `CC`: the compiler to be used
    * `SRC`: the `.c` files
